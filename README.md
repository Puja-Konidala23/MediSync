\# 🏥 MediSync – Smart Healthcare Management System



MediSync is a full-stack healthcare platform designed to connect patients, healthcare workers, and administrators in a unified system. It enables real-time resource tracking, efficient hospital management, and intelligent decision support for better healthcare outcomes.



\---



\## 🚀 Features



\- 👤 Multi-role Authentication (Admin / Worker / Patient)

\- 🏥 Hospital \& Resource Management (Beds, ICU, Oxygen)

\- 📅 Appointment Booking System

\- 🚨 SOS Emergency Alert System

\- 🤖 Intelligent Hospital Recommendations

\- 📊 Admin Dashboard with Analytics

\- 🔄 Real-time Data Synchronization

\- 📱 Responsive Design (Mobile + Desktop)



\---



\## 🛠️ Tech Stack



\- \*\*Frontend:\*\* React.js (TypeScript), Vite  

\- \*\*Styling:\*\* Tailwind CSS  

\- \*\*Animations:\*\* Framer Motion  

\- \*\*Charts:\*\* Recharts  

\- \*\*Backend \& Database:\*\* Firebase (Firestore, Authentication)  

\- \*\*API Handling:\*\* Axios  

\- \*\*AI Integration:\*\* Gemini API
